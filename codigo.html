O quê é pseudocódigo ?

Pseudocódigo é uma linguagem que simula linguagem de programação em português, As linguaguem de programação são escritos em English, mais em pseudocódigo os comandos são escritos em português. Para aqueles que estão iniciando a apreende o Algoritmo em pseudocódigo saiba que as instruções são escritos em português Atenção pseudocódigo não é uma linguagem de programação ele simula uma linguagem de programação, os camandos são escritos em português como o portugol. Para entender o este conceito é necessário que saibamos sobre algoritmo.


A programação em pseudocódigo são escrito em:
Algoritmo " "
Var
inicio
fimalgoritmo

Algoritmo: são sequência de passos lógico e ordenada para conseguir resolver um determinado problema. Devemos sempre escrever a função do nosso algoritmo dentro da virgula alta " ".

Var (variáveis): São espaço armazenado em memória do celular ou computador, onde declaramos a nossa variáveis. Variáveis podem ser declarado do tipo caracter, inteiro, real, lógico.

Início: Onde começamos a execução das variaveis que foram declarados no bloco var, todas os comandos são escritos dentro dele.

Fimalgoritmo: Devemos sempre fechar o nosso Algoritmo com um fimalgoritmo, que é o fechamento do algoritmo significa que nosso algoritmo acabou aqui.
