Aprenda criar site com celular

Para poderes ser programador web com celular é muito simples para aqueles pessoal que o interesse de ser desenvolvidor da página web.
Primeira coisa antes de tudo deves ter o domínio (conhecimento) sobre há linguaguem de marcação (HTML). O quê é uma linguagem de marcação HTML?
Uma linguagem de marcação de marcação HTML (Hyper text markup Languangue ou "linguagem de marcação de hipertexto") E o alcance fundamental da internet. não é uma linguagem de programação mais sim uma linguagem de marcação que organiza a estrutura o conteúdo do sites como o texto e imagens, vídeos, audios, formulário. essa é uma ferramenta fundamental da internet pôr que grandes indústrias possuem o endereço de um site para poder ser mais alcançado pelos clientes apartir de qualquer de lugar.

Para poder aprender é necessário que tenhamos o Editor Spock instalado no celular esses aplicativos esta disponível na PlayStore.
